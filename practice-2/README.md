## Getting started

- Clone repository

```
git clone https://github.com/ngocquang2509/traning-html-css.git
```

- Checkout branch develop

```
git checkout develop
```

- Change directory to practice-2 folder

```
cd practice-2
```

- Open cmd and install package

```
npm install
```

- Run project

```
npm start
```

- Open browser

```
http://localhost:1234
```
